# docx2md

Convert Microsoft Word Document to Markdown

## Usage

```
$ docx2md NewDocument.docx
```

## Installation

```
$ go get github.com/mattn/docx2md
```

## Supported Styles

* Header
* HyperLink
* Indent
* Table
* List
* Bold
* Italic
* Strike
